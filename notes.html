Mobile Development Errors & Issues

Install error/solutions for django and wagtail:

  • Django and Wagtail need tzdata to be installed via pip otherwise the admin sites will encounter internal server errors within a few minutes of starting.

Install error/solutions for streamlit:

  • PyArrow fix - install libarrow-cpp
  • Numpy fix - export $MATHLIB="m" then pip install streamlit
  • Streamlit fix - export LDFLAGS="-lpython3.11"

Install error solutions for pywebview:

  • pkg install pycairo pygobject and/or pip install pycairo pygobject
  • In .venv folder, change version of program if required. It could be different programs that cause the error if installed at different times due to termux only providing single versions of most programs. Check the error message for which ones to look for. The error i encountered couldn't import gi or find libc.6.so.
  • The webview needs to run via vncserver or xserver otherwise you will receive a failed to open display error. When creating an app that uses a GUI development is best done via Termux X11, chroot or proot to avoid this issue.
